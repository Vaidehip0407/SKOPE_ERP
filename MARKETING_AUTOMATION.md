## ðŸŽ¯ Campaign Creation

### Step-by-Step Process

1. **Campaign Details**
   - Name your campaign
   - Add description
   - Choose channel (WhatsApp/SMS/Email)
   - Select trigger type

2. **Message Template**
   - Use pre-built templates or create custom
   - Dynamic variables available:
     - `{customer_name}` - Customer's name
     - `{discount}` - Discount percentage
     - `{code}` - Discount code
     - `{days}` - Days before event
     - `{festival}` - Festival name
     - `{end_date}` - Offer end date
     - `{store_phone}` - Store contact

3. **Offer Configuration**
   - Set discount code (e.g., DIWALI50)
   - Set discount percentage
   - Configure validity period

4. **Scheduling**
   - Start date and time
   - End date and time
   - Preferred send time (e.g., 10:00 AM)
   - Days before trigger (for events)

5. **Target Audience**
   - All customers
   - Specific segments
   - Geo-location filtering
   - Custom criteria

## ðŸ’¡ Best Practices

### Timing
- **Birthday**: Send at 9:00 AM on birthday
- **Festival**: Start 2-3 days before festival
- **Warranty**: Alert 30 days before expiry
- **Cart Abandonment**: Send within 1-2 hours
- **Win-Back**: After 30 days of inactivity

### Message Content
- âœ… Keep it short and clear
- âœ… Use emojis for visual appeal
- âœ… Include clear call-to-action
- âœ… Add urgency (limited time)
- âœ… Personalize with name
- âŒ Avoid spam-like language
- âŒ Don't send too frequently

### Discount Strategy
- Birthday: 15-20% off
- Festival: 30-50% off
- Cart Recovery: 10-15% off
- Win-Back: 15-20% off
- Referral: 20-25% off (both parties)
- Warranty: Service discount or upgrade offer

### Testing
- Test campaigns with small audience first
- Monitor open and click rates
- Adjust message based on performance
- A/B test different offers
- Track conversion attribution

================================================================================
ROOT CAUSE:
================================================================================

The api.ts utility file already sets:
    baseURL: '/api/v1'

So when endpoints were defined as:
    endpoint: '/api/v1/reports/sales/product-wise'

The final URL became:
    /api/v1 + /api/v1/reports/sales/product-wise
    = /api/v1/api/v1/reports/sales/product-wise  ‚ùå (404 Error)

================================================================================
SOLUTION APPLIED:
================================================================================

Changed all endpoint definitions from:
    endpoint: '/api/v1/reports/...'

To:
    endpoint: '/reports/...'

Now the URLs are:
    baseURL (/api/v1) + endpoint (/reports/...)
    = /api/v1/reports/...  ‚úÖ (Works!)

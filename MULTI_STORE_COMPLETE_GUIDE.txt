API ENDPOINTS:
========================================================================

STORES MANAGEMENT (Super Admin only):
POST   /api/v1/stores/              - Create new store
GET    /api/v1/stores/              - List all stores
GET    /api/v1/stores/stats         - Get store statistics
GET    /api/v1/stores/{id}          - Get specific store
PUT    /api/v1/stores/{id}          - Update store
DELETE /api/v1/stores/{id}          - Deactivate store

GET /api/v1/inventory/products?store_id=1
GET /api/v1/sales/?store_id=1
GET /api/v1/customers/?store_id=1
GET /api/v1/financial/expenses?store_id=1
GET /api/v1/campaigns/?store_id=1
